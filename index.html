Dark Mode

Dark mode has multiple benefits such as eye strain reduction and battery saving while improving focus and enhancing readability.

Button

We used 2 arrays: darkmodeElements to make a collection of all the elements with .darkmode; darkmodeColors to store objects with 2 colors for "light and dark" modes for each item in darkmodeElements.

There are 6 functions in total:

  • toggleDarkmode() is paired with onclick event and it executes the other three toggle functions and changes the mode variable.
  • toggleNav() is used to toggle between modes on the nav by changing the classes.
  • toggleText() is used to toggle the text color of each HTML element with .text-darkmode to either white or black.
  • toggleBody() is used to toggle the background color of each HTML element with .darkmode to their responding colors in the array darkmodeColors.
  • changeImage() is paired with onclick event to change the image in the card on the left.
  • changeText() is paired with onclick event to change the text in the card on the left

The colors are stored in the array darkmodeColors as hex codes. There are objects in that array and each object has two values: the properties are dark and light; the values are the colors. The amount of objects and the amount of items in darkmodeElements are the same.

When the toggle button is clicked, it triggers toggleDarkmode() which triggers the rest of the toggles. To change the background color, we used for() loop and go through every item in darkmodeElements and sets it background color to the values of either dark or light from the objects stored in darkmodeColors with the use of setAttribute().

While creating this webpage, the major issue was thinking of how to make the website look good both in dark and light mode. Previously, we tried making classes with defined background colors and changing the class in HTML elements upon toggle. But was later changed to set colors in a list for dark and light modes in each element.
